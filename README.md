# Remapemanto-Diretorios-Windows
Script para Windows 10/11 que remapeia pastas do usuário (Docs, Desktop, etc.) para uma unidade secundária (D:/E:). Ideal para SSDs pequenos, automatiza o armazenamento de arquivos e mantém o sistema (C:) leve, sem necessidade de mover arquivos manualmente. Prático, rápido e multilíngue.
